# Calculadora em C

Esta é uma calculadora simples desenvolvida em linguagem C. 
O programa permite realizar operações matemáticas básicas, como adição, subtração, multiplicação, divisão e potenciação. Recebe a entrada do usuário, processa a operação desejada e exibe o resultado no terminal.
Este projeto foi criado com o objetivo de praticar e aprender mais sobre linguagem C.

## Como faço para rodar? 
É necessário seguir os seguintes passos:
- *IDE Visual studio code* instalado, link para [download](https://code.visualstudio.com/)
- Instalar o compilador [Mingw](https://www.youtube.com/watch?v=Mook-z2uvpM&pp=ygUiY29tbyBiYWl4YXIgY29tcGlsYWRvciBsaW5ndWFnZW0gYw%3D%3D) ou algum da sua escolha.
- Clone este repositório ou baixe os arquivos.
- Compile o código usando o seguinte comando: ``gcc calculadora.c -o calculadora -lm``
- Execute o programa usando o seguinte comando: ``./calculadora``


## Tecnologia Ultilizada

- **Linguagem C** — Linguagem de programação estruturada usada para desenvolver toda a lógica da calculadora.
- **math.h** — Biblioteca padrão da linguagem C, utilizada para realizar operações matemáticas como potenciação (`pow()`).
- **GCC (GNU Compiler Collection)** — Compilador utilizado para compilar o código, com o parâmetro `-lm` para linkar a biblioteca matemática.
- Logíca de programação: É um conjunto de técnicas e conceitos que permitem criar algoritmos para resolver problemas.
- Algoritmo: É uma sequência de instruções que, quando seguidas, permitem resolver um problema ou realizar uma tarefa.
